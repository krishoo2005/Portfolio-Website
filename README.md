##  Getting Started (Run Locally)

### Prerequisites:

* Node.js installed

### Steps:

1. Install dependencies

   ```bash
   npm install
   ```

2. Setup environment variables
   Create a `.env.local` file and add:

   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. Run the development server

   ```bash
   npm run dev
   ```
